# Rust AoC

Advent of code in Rust.

## Export session in a variable
  ```
  export AOC_SESSION=lksdklskdjf
  ```

## Create a day
   ```
   cargo new --name aoc-2018-<day> -- 2018/<day>
   ```

## Download a day
   ```
   cargo run --bin download <day>
   ```

## Run a day
   ```
   cargo run --bin aoc-2018-<day>
   ```
